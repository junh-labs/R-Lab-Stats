# R-Lab-Stats


## 项目简介
本仓库为**应用统计学 R 语言课程实验代码库**，完整收录实验1至实验5全部 R 脚本、配套数据集，同步配套实验指导文档，覆盖R入门、程序设计、数据预处理、可视化、描述性统计五大模块。

## 仓库目录结构

├── demo1/          # R语言入门演示代码

├── lab1/           # 实验1：R语言入门

├── lab2/           # 实验2：R语言程序设计

├── lab3/           # 实验3：R语言数据预处理

├── lab4/           # 实验4：R语言绘图可视化

├── lab5/           # 实验5：描述性统计分析

├── README.md       # 仓库说明文档


## 实验内容清单
1. **实验1 R语言入门**
   R基础语法、变量、向量、数据读写、基础数据结构操作
2. **实验2 R语言程序设计**
   分支循环、自定义函数、代码模块化编写
3. **实验3 R语言数据预处理**
   缺失值处理、数据集合并、数据清洗、噪声过滤
4. **实验4 R语言绘图**
   基础绘图、直方图、散点图、箱线图、多图拼接
5. **实验5 描述性统计分析**
   均值、方差、分位数、分布特征、基础统计指标计算

## 环境要求
- R ≥ 4.0
- 推荐编辑器：RStudio

## 使用方法
1. 克隆仓库到本地
```bash
git clone https://github.com/18725306273/R-Lab-Stats.git
```
2. 打开对应实验文件夹内 `.R` 脚本运行
3. 配套实验文档：各实验对应 `.docx` 实验指导书

## 更新代码流程
本地修改代码后，执行以下命令同步至GitHub：
```bash
git add .
git commit -m "更新说明"
git push
```


### 使用方式
1. 打开仓库页面，点击README右侧铅笔图标编辑
2. 全选原有文字删除，粘贴上面全部代码
3. 填写提交信息，确认提交即可刷新生效
